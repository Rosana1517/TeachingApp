Localhost 與 Port（連接埠）

電腦公寓理論，解決 EADDRINUSE 打架衝突

1 什麼是 Localhost？

Localhost 其實就是你的電腦自己（IP 是 127.0.0.1）。你的電腦就像一棟公寓，裡面有 65,535 個「房間」（這就是 Port / 連接埠）。

⚡ 為什麼多個任務會打架？

當你同時開兩個 Vibe 專案，它們都想擠進 3000 號房，就會噴出 EADDRINUSE 錯誤。

3 如何避免 Port 衝突？

學會「換房」技巧：npm run dev -- -p 3008，讓不同專案住在不同的房間，就能同時完美運行！

✏️ 小測驗

✏️ 小測驗

1. EADDRINUSE 錯誤代表什麼意思？
  • A. 網路斷線
  • B. Port 已被其他程式佔用
  • C. Node.js 版本太舊
✅ 正確答案：B。EADDRINUSE = Address already in use，代表你要使用的連接埠已經被另一個程式佔用了。
