# 教學 App - Neumorphism 組件展示

## 📱 視覺設計概覽

本文件展示教學 App 使用 Neumorphism（新擬態）設計風格的所有 UI 組件。

---

## 🎨 設計特色

### 視覺風格
- **柔和立體感**: 通過精細的陰影效果創造浮雕感
- **溫暖觸感**: 柔和的色彩和圓角帶來舒適的視覺體驗
- **層次分明**: 利用陰影深浅區分內容層次
- **流暢動畫**: 所有交互都有平滑的過渡效果

### 色彩方案
- **主色調**: 柔和的淺灰色背景 (#EDEFF4)
- **強調色**: 清新的藍紫色 (#667eea → #764ba2 漸層)
- **文字色**: 深灰色 (#1A1A2E) 確保可讀性
- **狀態色**: 成功綠、警告橙、錯誤紅

## ✨ 動畫效果

### 按鈕互動
1. **默認狀態**: 凸起陰影
2. **懸停狀態**: 陰影加深
3. **按下狀態**: 凹陷效果 + 縮放 0.95
4. **釋放**: 彈回默認狀態

### 卡片動畫
1. **載入**: 從下方滑入 + 淡入
2. **懸停**: 陰影略微加深
3. **點擊**: 輕微縮放反饋

### 頁面轉換
1. **前進**: 從右側滑入
2. **返回**: 向左側滑出
3. **淡入淡出**: 0.3秒過渡

## 🌓 深色模式

### 色彩映射

| 元素 | 淺色模式 | 深色模式 |
|------|---------|---------|
| 背景 | #EDEFF4 | #3D4045 |
| 凸起陰影（暗） | rgba(0,0,0,0.12) | rgba(0,0,0,0.4) |
| 凸起陰影（亮） | rgba(255,255,255,0.75) | rgba(255,255,255,0.05) |
| 主要文字 | #1A1A2E | #FFFFFF |
| 次要文字 | #737780 | #B2B4B8 |

### 注意事項
- 深色模式下陰影對比度更高
- 確保文字可讀性
- 保持一致的設計語言
