# 參考資源清單

## Node.js 與 JavaScript 執行環境

- [Node.js 官方入門指南](https://nodejs.org/en/learn/getting-started/introduction-to-nodejs) — 最權威的 Node.js 介紹，涵蓋 V8、Event Loop、模組系統等核心概念。
- [MDN: JavaScript 引擎工作原理](https://developer.mozilla.org/en-US/docs/Glossary/V8) — 深入了解 V8 的編譯與優化機制。
- [Node.js 官方文件](https://nodejs.org/docs/latest/api/) — API Reference，查閱所有內建模組。
- [Understanding Event Loop, Callbacks, Promises and Async/Await in JavaScript](https://www.freecodecamp.org/news/understanding-javascript-event-loop-promises-and-async-await/) — freeCodeCamp 的事件循環深度文章。

## NPM 與套件管理

- [NPM 官方文件](https://docs.npmjs.com/) — package.json、dependencies、scripts 的完整說明。
- [npm create](https://docs.npmjs.com/cli/v10/commands/npm-create) — npx 與專案初始化工具的使用方式。

## Next.js
